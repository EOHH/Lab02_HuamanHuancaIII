Conclusions

  1. We managed to give content to the pages in each exercise, so that the layout and presentation of the content has a good interaction with visitors, this will also allow a better visualisation for visitors and good navigation.
  2. We managed to group the pages of the 5 exercises in a single page, from there you can navigate to the different pages and content.
  3. The javascript programming language was used to make our page interactive.
  4. The "var" tag was used to declare variables, this depends on how we want to set our variables, be it text or numeric.
  5. The tags "alert" and "document.write" were used to show the results of the exercises.
  6. We used CSS language to give design and responsiveness to our pages, we added page colour, text colour, text size, text position, among other functionalities.
